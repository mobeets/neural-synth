Neural synth

Using variational autoencoders to make probabilistic music.

Variational autoencoders are generative neural networks that learn how to generate music using a low-dimensional latent space. If we map the latent space to a pad controller (as below), clicking on the pad tells the autoencoder to output the notes corresponding to that pad position. Because the output of the autoencoder is probabilistic, you won't get exactly the same notes every time.

This network was trained on 382 of Bach's four-part chorales transposed to Cmaj or Cmin (source). Code and more information is available here.
